# 黄金价格与波动率监控系统

实时监控黄金价格和波动率的 Python 全栈应用。

## 功能特性

- 🔄 自动抓取实时黄金价格数据（支持多种API，包含yfinance备选）
- 📈 计算日波动率、周波动率
- 💾 本地SQLite数据库存储历史数据
- 📊 可视化展示价格走势和波动率曲线
- 💻 简洁易用的Web界面（原Streamlit，现推荐Dash）
- 📤 支持数据刷新、查询、导出

## 快速开始

### 环境要求
- Python 3.8 ~ 3.11
- Windows / macOS / Linux

### 安装依赖
```bash
pip install -r requirements.txt

# 如果使用Dash前端，还需要安装对应依赖：
pip install dash dash-bootstrap-components yfinance
```

> 注意：requirements.txt 中已包含 akshare 等必需库，确保安装成功以避免部署时出错。

### 运行程序
```bash
# 运行 Streamlit 界面（保留备份）
python main.py

# 或者运行 Dash 界面
python dash_app.py
```

程序启动后，在浏览器中打开 `http://localhost:8501` 查看界面。

## 核心功能模块

### 1. 数据抓取模块
- 使用akshare库抓取上海黄金交易所数据
- 支持历史数据批量导入和实时数据更新
- 自动去重和异常值过滤

### 2. 数据计算模块
- 计算收益率（对数收益率）
- 计算日/周波动率
- 计算涨跌幅和涨跌额
- 计算移动平均线

### 3. 数据存储模块
- SQLite3本地数据库
- 支持导出为Excel/CSV

### 4. 前端界面
- 实时显示黄金价格和技术指标
- 支持Streamlit（旧版）和Dash（推荐）
- K线图、波动率走势图
- 历史数据查询和导出

## 使用说明

### 首次运行
程序会自动：
1. 创建数据库表
2. 抓取历史数据（5分钟级别）
3. 启动Web服务

### 数据更新
程序会每5分钟自动更新一次最新数据（可配置）。
手动刷新按钮可立即更新。

> ⚠️ 修复：当无法访问外部API时，系统不再使用错误的500元/克默认值，改为1100元/克并尝试通过yfinance获取当前价格。

### 数据导出
在Web界面点击"导出数据"按钮可导出为Excel文件。

## 核心计算公式

### 1. 对数收益率
```
收益率 = ln(今日收盘价 / 昨日收盘价)
```

### 2. 日波动率
```
日波动率 = 收益率序列的标准差
```

### 3. 周波动率
```
周波动率 = 日波动率 × sqrt(5)
```

### 连接错误

程序会在下一个更新周期自动补全缺失数据。

### 性能问题
如果数据量过大，可在config中调整分页参数。
